---- Installation Instructions ----
1. Download python3 from the windows store
2. Open Command Prompt and run:
    - $ pip3 install numpy
    - $ pip3 install pandas
    - $ pip3 install tk

---- Usage Instructions wpt_convert.py ----
1. In command prompt, navigate to this directory ($ cd filepath)
2. In command prompt, run:
    $ python3 wpt_convert.py
3. The program will prompt you to open a file, open a .l84 file exported from Qinsy
4. In the WAM-V waypoints editor (in our gui) press the open button and open the output.wpt file
